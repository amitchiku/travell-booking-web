# 🧭 Amitchiku Travel Agency

A premium, modern, and fully responsive travel agency web application built using **React**, **Vite**, **Tailwind CSS**, and **Firebase**.
The platform allows users to explore destinations, discover travel packages, book trips online, and manage bookings seamlessly with a clean and interactive user experience.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/amitchiku/amitchiku-travel-agency.git
```

---

```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_API_KEY=your_firebase_api_key
VITE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_PROJECT_ID=your_project_id
VITE_STORAGE_BUCKET=your_project.appspot.com
VITE_MESSAGING_SENDER_ID=your_sender_id
VITE_APP_ID=your_app_id
```

---

## 5️⃣ Start Development Server

```bash
npm run dev
```

# 🔥 Firebase Setup

1. Create a Firebase Project
2. Enable Firestore Database
3. Register a Web Application
4. Copy Firebase credentials into `.env`
5. Create Firestore Collections:

   * `bookings`
   * `enquiries`
